# 😎 MLL Bot

**MLL Bot** is an **all-in-one Discord bot** combining **Moderation, Automation, Economy, Leveling, Dashboard, Fun & Games, and Advanced Features** in a single bot.

---

## 📌 About

MLL Bot is designed for **scalable, modular, and production-ready Discord server management**.  
It includes moderation, economy, leveling, auto systems, dashboard, fun commands, and more.

---

## 🛠 Core Competencies

- ✅ Full Moderation (Ban/Kick/Mute/Role/Warnings)  
- ✅ Auto Reply / React / Line Automation  
- ✅ Economy System (Credits, Leaderboard, Tax, Transfers)  
- ✅ Leveling System (Chat & Voice XP, Rank, Profile)  
- ✅ Ticketing / Support System  
- ✅ Logging & Audit System  
- ✅ Custom Commands & Reaction Roles  
- ✅ Fun & Games (Trivia, Tic Tac Toe, Hangman, Coin Flip)  
- ✅ Persistent Mute & Auto Role  
- ✅ Music / Audio Features  
- ✅ Reminders / Scheduled Messages  
- ✅ Server Stats / Insights

## 🎮 Features

### Moderation
- Ban / Kick / Mute / Timeout / Role management  
- Warn system  
- Logging & Audit  

### Automation
- Auto Reply / Auto React / Auto Line  
- Reaction Roles / Self Roles  

### Economy
- Credits / Leaderboard / Tax / Transfers  
- Level-up rewards and customizable economy  

### Leveling
- XP from chat & voice  
- Profile cards and Rank leaderboard  

### Dashboard
- Web management of all bot settings  
- Prefix, permissions, channels, economy, auto systems  

### Fun & Games
- Trivia / Hangman / Tic Tac Toe / Coin Flip / Dice  

### Music & Audio
- Play YouTube / Spotify tracks  
- Pause / Resume / Skip / Queue  

### Tickets & Reminders
- Support ticket system  
- Scheduled reminders and messages  

### Stats & Insights
- Server stats, daily messages, voice activity  
- Graphs on Dashboard  

---

## 🔐 Security
- Works only in allowed servers (`ALLOWED_SERVERS`)  
- Automatically leaves unauthorized servers  
- Persistent mute and auto roles
